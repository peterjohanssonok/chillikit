# Chilli PHP Kit 🌶️

A lightweight PHP application kit for learning, prototyping, and building practical web applications.

## What can I build with Chilli?

- Personal websites
- Blogs
- Internal business tools
- Customer portals
- Documentation sites
- Prototypes
- Small CMS solutions
- Learning projects

Build complete PHP applications using only PHP and SQLite and learn how applications work internally before using a framework.
